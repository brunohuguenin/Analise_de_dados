Este repositório reúne os conteúdos, exercícios e projetos desenvolvidos ao longo do curso de Análise de Dados e Big Data.
O objetivo principal é consolidar os fundamentos da área, aplicando conceitos teóricos na prática por meio de códigos, análises e manipulação de dados.

Conteúdos abordados até o momento:
Algoritmos e Estrutura de Dados com Python

1 - Lógica de programação aplicada à análise de dados
- Estruturas básicas (listas, tuplas, dicionários, conjuntos)
- Controle de fluxo (condicionais e laços)
- Funções e organização de código
- Introdução à análise de complexidade e boas práticas

2 - Análise de Dados com Pandas
- Manipulação e limpeza de dados
- Leitura e escrita de arquivos (CSV, Excel, etc.)
- Tratamento de dados ausentes e inconsistências
- Filtragem, ordenação e agregações
- Análises exploratórias básicas com DataFrames

3 - Banco de Dados e SQL
- Conceitos fundamentais de banco de dados
- Consultas SQL (SELECT, WHERE, JOIN, GROUP BY, ORDER BY)
- Extração de dados para análise
- Integração entre SQL e Python

Objetivo do repositório:
Registrar a evolução no aprendizado em Análise de Dados;
Servir como material de consulta futura;
Demonstrar prática com Python, Pandas e SQL;
Construir uma base sólida para temas mais avançados em Big Data;
